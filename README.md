## Lokaal Installeren en Draaien

1.  **Installeer de dependencies:**
    ```bash
    npm install
    ```
2.  **Start de ontwikkelserver:**
    ```bash
    npm run dev
    ```
De applicatie draait nu op `http://localhost:5173`.
